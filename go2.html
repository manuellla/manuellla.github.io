Quadrupedal Autonomous Exploration

Developing a Robust Autonomy Stack for the Unitree Go2

Research Overview

This project represents my current research focused on building a comprehensive autonomous navigation stack for the Unitree Go2 quadruped robot. Currently a work in progress, the development utilizes ROS 2 Humble and Gazebo Classic to create a high-fidelity simulation environment that bridges the gap between digital testing and real-world exploration. A quick demo of the autonomous waypoint following: here.

The long-term goal of this research is to explore adaptive and shared autonomy, enabling the robot to navigate complex, unstructured environments with increasing independence while maintaining a collaborative loop with human operators.

Technical Architecture

  • Locomotion Control: Usage of the CHAMP controller to manage stable gait patterns and dynamic quadrupedal walking.
  • Perception & SLAM: Integration of a Livox Mid360 LiDAR sensor, utilizing FAST-LIO2 for high-frequency odometry and precise 3D mapping.
  • State Estimation: A multi-sensor fusion approach using an Extended Kalman Filter (EKF) via the Robot Localization package to stabilize the global transform by integrating SLAM data, IMU, and robot odometry.
  • Mapping & Navigation: Real-time generation of 2D occupancy maps using Octomap Server, with mission logic powered by the Nav2 (Navigation 2) stack for path planning and obstacle avoidance.

Current Status & Roadmap

The system currently supports stable waypoint following and "patrol mode" within local Cartesian coordinates. The immediate roadmap involves the integration of GNSS (Global Navigation Satellite System) data to transition from local simulation coordinates to world-scale outdoor navigation. This advancement is a key step toward the final objective of deploying the autonomy stack on the physical Unitree Go2 for real-world environmental exploration. Currently, we are implementing our stack in the real robot as well, so results are coming soon!
